IQRA FUTURE GENERATION – SALAH JOURNEY

এটি সবুজ থিমের একটি প্রস্তুত static web app।

দ্রুত পরীক্ষা:
1. index.html ফাইলটি কম্পিউটারে Chrome দিয়ে খুলুন।
2. Kinderansicht থেকে বাচ্চার নাম নির্বাচন করুন।
3. সালাতের বক্সে ট্যাপ করে ⭐ দিন।
4. Lehreransicht-এর Demo PIN: 2026

অনলাইনে দেওয়ার সহজ উপায় (Netlify Drop):
1. এই ZIP ফাইলটি unzip করুন।
2. কম্পিউটারে https://app.netlify.com/drop খুলুন।
3. iqra_salah_journey ফোল্ডারটি drag & drop করুন।
4. Netlify একটি public link দেবে।

গুরুত্বপূর্ণ সীমাবদ্ধতা:
এই version-এর তথ্য প্রতিটি browser/device-এর local storage-এ আলাদাভাবে save হয়।
অর্থাৎ বিভিন্ন বাড়ি থেকে বাচ্চাদের data শিক্ষক একই dashboard-এ দেখতে পারবেন না।
সেটির জন্য পরবর্তী version-এ online database, login এবং secure user accounts লাগবে।
